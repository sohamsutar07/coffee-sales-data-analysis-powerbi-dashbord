☕ Coffee Sales Data Analysis Project | Excel, SQL, Python, Power BI

Excited to share my end-to-end Data Analyst project where I analyzed a coffee shop sales dataset to uncover meaningful business insights and visualize key sales trends.

In this project, I worked with multiple data analytics tools to clean, analyze, and visualize sales data, ultimately creating an interactive Power BI dashboard to support data-driven decision making.

🧠 Project Objective
The goal of this project was to analyze coffee sales data to understand:
 • Product performance
 • Customer payment behavior
 • Sales trends over time
 • Revenue distribution across different coffee types

🛠 Tools & Technologies Used
 • Microsoft Excel – Data cleaning and preparation
 • SQL – Business queries and sales analysis
 • Python (Pandas, Matplotlib) – Exploratory Data Analysis (EDA) and   visualizations
 • Power BI – Interactive dashboard and data visualization

📊 Key Analysis Performed
 • Cleaned and prepared the dataset for analysis using Excel
 • Wrote SQL queries to calculate total sales, product-wise sales, and payment analysis
 • Performed EDA in Python to explore trends and sales distribution
 • Built a Power BI dashboard to visualize key business insights

📌 Dashboard Highlights
  The Power BI dashboard includes:
 • Total Sales KPI
 • Product-wise Sales Analysis
 • Monthly Sales Trend
 • Payment Method Distribution (Cash vs Card)
 • Daily Sales Trend with interactive filters

💡 Key Insights
• Latte emerged as the highest selling coffee product
 • Card payments were used more frequently than cash
 • Sales trends showed variation across different dates and months
 • Visualization helped identify top-performing products and sales patterns

 • Data cleaning and preparation

 • Performing data analysis using Python
 • Building interactive dashboards in Power BI
 • Communicating insights through data visualization
